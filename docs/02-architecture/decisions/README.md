# Architecture Decision Records (ADRs)

Index of Architecture Decision Records. Each ADR captures a significant design
decision, its context, and its consequences.

**Before changing any non-trivial subsystem, read its ADR first.** These records
carry "don't reintroduce X" regression guards for bugs that were already fixed
once — they exist to stop the same mistake from being made twice.
